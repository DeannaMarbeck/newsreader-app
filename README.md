# Newsreader App

## A mobile app built using React Native, Redux, Axios and thunks.

Pulls news headlines from the News API https://newsapi.org/, displays the truncated headlines in a list view, allows clickthrough to a detail page, and links to the full article on the BBC news site. Built after learning about APIs and mobile apps using React Native.

To view the app, go to this link: https://exp.host/@deannamarbeck/newsreader-app-dmarbeck

You will need to have the Expo mobile client installed.
Documentation for the client here: https://expo.io/tools#client
